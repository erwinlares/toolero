# toolero: A Toolkit for Research Workflows

Provides utility functions to help researchers implement best practices
for their coding projects. Includes tools for reading and cleaning data
files, initializing R projects with a standard folder structure,
creating Quarto documents from a reproducible template, detecting the
execution context across interactive, Quarto, and script-based
workflows, and splitting data frames into group-level output files.
